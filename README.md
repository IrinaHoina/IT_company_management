# Домашнее задание: Дерево метаданных

## Тема занятия
Создание объектов метаданных в 1С:Предприятие 8.3

## Цель работы
Научиться создавать справочники, перечисления, функциональные опции и подсистемы в конфигурации "УправлениеИТФирмой".

---

## Результат выполнения
Файл конфигурации `.cf` (или выгрузка в файлы), содержащий все созданные объекты:
- Справочник "Сотрудники"
- Справочник "Контрагенты"
- Перечисление "ЮридическоеФизическоеЛицо"
- Подсистема "Справочники"
- Функциональная опция "ВестиРасчетЗарплаты"
- Константа "ВестиРасчетЗарплаты"

---

## Задача 1. Справочник "Сотрудники"

### Требования
Создать справочник "Сотрудники" и функциональную опцию "ВестиРасчетЗарплаты".

### Состав реквизитов справочника
| Реквизит | Тип | Примечание |
|----------|-----|------------|
| Email | Строка (100) | Электронная почта |
| Телефон | Строка (20) | Номер телефона с учетом кода страны |
| Оклад | Число (15,2) | Использовать определяемый тип "Сумма" |
| СтавкаЧаса | Число (15,2) | Использовать определяемый тип "Сумма" |

### Функциональная опция
- **Имя:** ВестиРасчетЗарплаты
- **Тип:** Булево
- **Состав:** реквизиты `Оклад` и `СтавкаЧаса`

### Константа
- **Имя:** ВестиРасчетЗарплаты
- **Тип:** Булево
- **Значение по умолчанию:** Ложь

### Проверка результата
1. Запустить конфигурацию в режиме "Предприятие"
2. Открыть справочник "Сотрудники" — поля "Оклад" и "СтавкаЧаса" скрыты
3. Через "Все функции" найти константу "ВестиРасчетЗарплаты" и установить = Истина
4. Снова открыть справочник — поля должны появиться

## Задача 3. Подсистема "Справочники"

### Требования
Создать подсистему, объединяющую созданные справочники.

### Параметры подсистемы
- **Имя:** Справочники
- **Картинка:** выбрать подходящую (например, "Справочник")
- **Состав:** включить справочники "Сотрудники" и "Контрагенты"

### Проверка результата
1. Запустить конфигурацию в режиме "Предприятие"
2. В командном интерфейсе должен появиться раздел "Справочники"
3. В разделе должны быть команды для открытия справочников "Сотрудники" и "Контрагенты"

---

## Как открыть мою конфигурацию

### Способ 1 (если выгружено в файлы)
1. Создать пустую информационную базу
2. Конфигуратор → Конфигурация → Загрузить конфигурацию из файлов
3. Выбрать папку с этой выгрузкой
4. Нажать F7 (Обновить конфигурацию базы данных)
5. Запустить 1С:Предприятие (F5)

### Способ 2 (если передан файл .cf)
1. Создать пустую информационную базу
2. Конфигуратор → Конфигурация → Загрузить конфигурацию из файла
3. Выбрать файл `.cf`
4. Нажать F7 → F5
